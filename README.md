# ApiCep_java

Este é um sistema simples em Java que realiza consultas de CEP usando uma API pública.
Ele utiliza POO (Programação Orientada a Objetos) e inclui tratamento de exceções para lidar com CEPs inválidos ou erros na consulta.
O sistema permite:

- Buscar endereços pelo CEP

- Excluir endereços cadastrados

- Consultar histórico de CEPs digitados
